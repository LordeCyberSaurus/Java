Escreva um programa Java que usa tratamento de erros try...catch para forçar o usuário a informar dois valores inteiros válidos antes de tentar somá-los. Caso o usuário informe um ou os dois valores inválidos, o programa deverá exibir uma mensagem de erro e solicitar os valores novamente.
Em Java, um exceção do tipo java.lang.ArithmeticException: / by zero (divisão por zero) é causada quando tentamos efetuar uma divisão usando inteiros e o divisor é 0. Escreva um programa que solicita ao usuário dois números inteiros e efetue a divisão do primeiro pelo segundo. Trate a exceção (erro) java.lang.ArithmeticException: / by zero para evitar que o programa termine caso o usuário informe 0 para o segundo valor.
